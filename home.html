How It Works

Your code goes through 4 compilation stages. Each stage's output is displayed in real-time on the dashboard.

1

Lexer

Breaks source code into tokens — keywords, numbers, operators, and identifiers.

2

Parser

Builds an Abstract Syntax Tree (AST) respecting operator precedence and block structure.

3

Generator

Translates the AST into low-level assembly instructions like MOV, ADD, CMP, JMP.

4

Executor

Runs the program and displays the final output — the result of your code.
